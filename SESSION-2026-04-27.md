## Notable decisions

- **Spec deviation: Next 14.2.35 instead of 14.2.18.** The original CLAUDE.md called out no specific Next version, but `npm install next` resolved to `14.2.18` which has a published CVE (security update 2025-12-11). Pinned to the patched 14.2.35 release line.
- **Spec deviation: `@next/font` omitted.** Folded into the built-in `next/font` in Next 13.2. Installing the legacy package only pulls a deprecation shim.
- **`ReferralCodeBox` displays the bare code.** The component spec said "URL", the homepage spec said `STAR-GCQJ-N6NC`. Resolved by displaying the code as the visible centerpiece and adding a `copyValue="url"` prop for callers that want the full URL on the clipboard.
- **Screenshot placeholders on `/how-to-use`** are styled HUD-frame cards with captions, not real images. Easy swap once real screenshots are produced.
- **Footer** includes a sister-site link to `screferralrewards.com`. Not in the original spec but consistent with the portfolio relationship described in the project Overview.
